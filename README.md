## **项目名称：RSA 文本加密与图像加密技术实验**

本项目实现了 RSA 文本加密解密流程，以及基于 XOR 的图像加密与密文运算。项目旨在帮助理解传统密码学、图像加密和密文可计算性在隐私保护中的基础作用，并展示相关技术在云计算与联邦学习等前沿领域的潜在应用。

---

## **项目结构**

```
├── rsa_demo.py                # RSA 公钥加密与私钥解密示例程序
├── image_encrypt_demo.py      # 图像加密、密文运算与解密示例程序
├── input1.png                 # 实验图片1（灰度图）
├── input2.png                 # 实验图片2（可选，用于密文 XOR）
└── output_images/             # 程序运行生成的输出图像
```

---

## **环境依赖**

请先安装以下 Python 包：

```bash
pip install cryptography pillow numpy
```

环境：

* Python ≥ 3.8
* 支持 OpenSSL 的系统环境（Windows / Linux / macOS 均可）

---

## **一、RSA 文本加密实验**

### **功能说明**

* 自动生成 RSA 公钥与私钥
* 使用公钥加密文本
* 使用私钥解密恢复原文
* 支持 PEM 格式密钥保存与加载

### **运行方式**

```bash
python rsa_demo.py
```

### **实验结果**

程序会输出：

* 生成的公钥与私钥
* 文本加密后的密文（十六进制）
* 解密还原的明文

---

## **二、图像加密与密文运算实验**

### **功能说明**

* 将灰度图像读取为像素矩阵
* 为图像生成对应尺寸的随机密钥
* 使用 XOR 完成图像加密与解密
* 支持密文状态下的：

  * 统计运算（均值、方差）
  * 两幅加密图像的 XOR 运算（跨图像密文计算）

### **运行方式**

```bash
python image_encrypt_demo.py
```

### **输出结果**

程序将在 `output_images/` 文件夹中生成：

* `cipher_image1.png`：图像1加密后
* `cipher_image2.png`：图像2加密后（若存在）
* `decrypted_image1.png`：解密还原图像
* `cipher_xor.png`：两幅密文图像 XOR 后的密文结果

这些图像可用于报告展示与分析。

## **项目亮点**

### **✔ RSA 非对称加密的完整实现**

掌握密钥生成、加密、解密、密钥文件保存与加载的全流程。

### **✔ 图像加密的基础实现**

以 XOR 作为轻量级图像加密方案，实现密文不可读性与加密可逆性。

### **✔ 密文可计算性演示（项目核心）**

实现密文图像之间的 XOR 运算，展示加密数据无需解密仍可参与计算，为理解同态加密、隐私计算奠定基础。

---

## **应用前景**

该项目展示的加密与密文运算机制，在以下领域具有重要价值：

* **云计算隐私保护**：云端可对加密图像直接进行部分处理，无需明文暴露。
* **联邦学习**：加密图像可参与模型训练与推理，保护数据提供方隐私。
* **医疗数据安全共享**：支持加密医疗影像的跨机构分析与匹配。
* **智能监控与边缘计算**：终端上传加密图像，降低隐私泄露风险。
